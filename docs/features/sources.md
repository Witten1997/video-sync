# 视频源管理

## 支持的源类型

### 收藏夹
监控收藏夹更新，自动下载新增视频。

**添加方式**：
1. 进入「任务源管理」
2. 选择「收藏夹」类型
3. 输入收藏夹ID（从URL获取）
4. 配置过滤规则（可选）

### UP主投稿
追踪UP主所有投稿。

**添加方式**：
1. 选择「UP主投稿」类型
2. 输入UP主ID（从主页URL获取）
3. 配置过滤规则

### 视频合集
下载整个合集的所有视频。

**添加方式**：
1. 选择「合集」类型
2. 输入合集URL
3. 配置过滤规则

### 单个视频
快速下载单个视频。

**添加方式**：
进入「快捷订阅」，粘贴视频URL。

## 过滤规则

### 时长过滤
```
最小时长：60秒
最大时长：3600秒
```

### 分区过滤
选择允许的分区：
- 动画、番剧
- 国创、音乐
- 舞蹈、游戏
- 知识、科技
- 运动、汽车
- 生活、美食
- ...

### 标题过滤
- **包含关键词**：仅下载标题含指定词的视频
- **排除关键词**：跳过标题含指定词的视频

## 管理操作

- **启用/禁用**：控制是否同步该源
- **立即同步**：手动触发单个源同步
- **编辑**：修改配置和过滤规则
- **删除**：移除任务源（不删除已下载文件）
