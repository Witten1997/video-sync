# 简介

Video-Sync 是专为 NAS 用户设计的哔哩哔哩视频同步工具，自动化下载与管理 B 站视频。

## 核心功能

### 视频同步
- 收藏夹自动同步
- UP主全部投稿追踪
- 视频合集批量下载
- 单个视频快速下载
- 定时扫描和自动下载

### 下载管理
- 断点续传
- 智能过滤（按时长、分区、标题）
- 质量选择（最高1080P+）
- 并发控制

### 媒体库集成
- NFO 元数据自动生成
- 视频封面和UP主头像下载
- 标准化文件命名
- 与 Emby/Jellyfin/Plex 无缝对接

### 弹幕功能
- XML弹幕下载
- 自动转ASS字幕格式
- 支持弹幕样式自定义

### 管理界面
- 现代化Web UI
- 实时任务监控
- 扫码登录
- 在线配置管理
- 数据统计

## 技术栈

**后端**
- Go 1.23.3+ + Gin
- PostgreSQL 16+
- yt-dlp + FFmpeg

**前端**
- Vue 3 + Vite 5
- Element Plus
- ECharts + video.js
