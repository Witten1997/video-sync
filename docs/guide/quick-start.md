# 快速开始

## Docker 部署（推荐）

### 1. 准备配置

创建目录：
```bash
mkdir -p downloads metadata configs logs
```

### 2. 启动服务

```bash
docker-compose up -d
```

### 3. 访问界面

浏览器打开 `http://localhost:21001`

### 4. 登录配置

1. 进入「配置中心」
2. 点击「扫码登录」
3. 使用 B 站 APP 扫码
4. 自动获取登录凭证

### 5. 添加任务源

进入「任务源管理」，添加：
- 收藏夹：输入收藏夹ID
- UP主：输入UP主ID
- 合集：输入合集URL

### 6. 开始同步

- 手动触发：点击「立即同步」
- 自动同步：系统按配置间隔自动扫描

## 验证部署

访问仪表盘查看：
- 任务统计
- 下载进度
- 系统状态
